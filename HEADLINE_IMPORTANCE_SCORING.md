# 头条重要性评分系统

## 问题

之前的排序只看**发布时间**，导致"对老虎禁食"这种琐事新闻排在前面，而重要的战争、政治新闻可能被埋没。

## 解决方案

新的排序算法：
```
综合评分 = 内容重要性(60%) + 发布时间(30%) + 标题质量(10%)
```

## 内容重要性评分（0-100分）

### 高重要性关键词（+分）

#### 战争/军事（最高优先级）
- **+30分**：war, 战争, войн
- **+25分**：attack, missile, bombing, invasion, 袭击, 导弹, 轰炸, атак, ракет
- **+20分**：strike, military, combat, 军事, военн
- **+15分**：weapon

#### 政治重大事件
- **+20分**：coup (政变)
- **+15分**：president, election, treaty, sanction, resign, 总统, 选举, 制裁, 辞职, президент, санкц
- **+10分**：government, parliament, 政府, правительств

#### 经济重大事件
- **+20分**：collapse, 崩溃, крах
- **+15分**：crisis, bankruptcy, recession, 危机, 破产, 衰退, кризис, банкротств

#### 灾难事故
- **+20分**：disaster, earthquake, explosion, 灾难, землетрясен, взрыв
- **+15分**：flood, crash, 死亡
- **+10分**：fire

### 低重要性关键词（-分）

#### 娱乐八卦
- **-20分**：gossip, 八卦
- **-15分**：celebrity, fashion, beauty, 明星, 时尚, 美容

#### 日常琐事
- **-10分**：tips, advice, how to, guide, 建议, 指南, 技巧, совет, рекоменд

#### 动物趣闻
- **-15分**：zoo, cute, adorable, 动物园, 可爱, зоопарк
- **-10分**：pet, 宠物, питомц

#### 生活琐事
- **-20分**：horoscope, 星座, гороскоп
- **-15分**：recipe, 食谱, рецепт
- **-10分**：diet, workout, 饮食

### 特殊规则

- **标题太短**（<20字符）：-10分
- **包含数字**：+5分（更具体）

## 实际案例分析

### 案例1：老虎禁食新闻

```
标题：俄罗斯一动物园在节日后对老虎实施间歇性禁食
关键词：动物园(zoo), 老虎
评分：50（基础分）- 15（zoo）= 35分
排名：靠后 ✓
```

### 案例2：导弹袭击新闻

```
标题：乌克兰向距离边境800公里的俄罗斯地区发射了新型导弹
关键词：导弹(missile), 袭击, 800公里
评分：50（基础分）+ 25（missile）+ 5（数字）= 80分
排名：靠前 ✓
```

### 案例3：巴基斯坦-阿富汗战争

```
标题：巴基斯坦宣布与阿富汗进入"公开战争"状态并对喀布尔发动空袭
关键词：战争(war), 空袭(attack)
评分：50 + 30（war）+ 25（attack）= 105分 → 限制为100分
排名：最前 ✓
```

### 案例4：美容程序新闻

```
标题：一名俄罗斯女性在接受一项流行美容程序后，展示了其布满伤口的面部
关键词：美容(beauty)
评分：50 - 15（beauty）= 35分
排名：靠后 ✓
```

## 综合评分公式

```python
综合评分 = (
    内容重要性 × 0.6 +
    发布时间归一化 × 0.3 +
    标题质量 × 0.1
)
```

### 权重说明

- **内容重要性 60%**：最重要，决定新闻价值
- **发布时间 30%**：保证时效性
- **标题质量 10%**：辅助判断

## 调优建议

### 如果重要新闻还是被埋没

```python
# 提高内容重要性权重
综合评分 = (
    内容重要性 × 0.7 +  # 从60%提高到70%
    发布时间 × 0.2 +    # 从30%降到20%
    标题质量 × 0.1
)
```

### 如果需要更重视时效性

```python
# 提高发布时间权重
综合评分 = (
    内容重要性 × 0.5 +
    发布时间 × 0.4 +    # 从30%提高到40%
    标题质量 × 0.1
)
```
